## 🚀 Featured Projects

### 💬 [ChatHub](https://github.com/ghurstird/chatHub)
A unified messaging app that brings **WhatsApp, Telegram, Twitter, Instagram, and Bluesky** into a single inbox, built on the **Matrix protocol**. Real-time messaging with media support (images, video, voice) and push notifications.
I contributed across **all stages** of the project, from design to implementation.

`React Native (Expo)` · `Node.js / Express` · `Matrix (matrix-js-sdk)` · `WebSockets` · `Firebase`

### 🧠 [turkish-embedding-benchmark](https://github.com/wolixistaken/turkish-embedding-benchmark)
Benchmarks Turkish/multilingual **embedding models** on an LLM Q&A dataset — comparing GPT-4o vs DeepSeek answers through retrieval (Top-1 / Top-5 accuracy) and classification.

`PyTorch` · `sentence-transformers` · `scikit-learn`

### 📈 [cnn-image-regression-optuna](https://github.com/wolixistaken/cnn-image-regression-optuna)
A **CNN regression** study on synthetic images, using **Optuna** for hyperparameter optimization across multiple geometric tasks and training-set sizes.

`PyTorch` · `Optuna` · `NumPy`
